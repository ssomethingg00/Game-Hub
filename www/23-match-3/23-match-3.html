How to Play
  • 👆Tap a gem to select it, then tap an adjacent gem to swap them.
  • ↔️Or drag/swipe a gem toward its neighbor. Works on mobile and desktop.
  • 💎Line up 3 or more gems of the same color + shape to clear them. Gems fall down, new ones drop in.
  • ⚡4 in a line → Line Gem (clears whole row or column). L/T shape → Burst Gem (3×3 explosion). 5 in a line → Nova Gem (clears all of one color!).
  • 🎯Reach the target score before moves run out. Each level raises the target and starts with a fresh board.
  • 🔗Cascades (chain reactions) multiply your score — bigger chains = bigger points!
  • ♿Each gem has a unique color + shape symbol so you never need to rely on color alone.
