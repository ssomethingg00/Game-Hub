How to Play

👆

Flip Two Cards

Tap (or click) any two face-down cards to reveal their symbols.

🎯

Find Matching Pairs

If both cards show the same symbol, they stay face-up — that's a match! If not, both flip back.

⭐

Earn Stars

Fewer moves = more stars. Match all pairs in under 1.5× the pair count for 3 stars.

🔥

Build Streaks

3 consecutive matches without a mismatch earns an "On Fire!" combo bonus.

🏆

Unlock Achievements

Complete challenges like a Perfect Game (no mismatches) or speed runs for special badges.
